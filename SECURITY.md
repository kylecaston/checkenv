# Security Policy

## Supported Versions

`checkenv` follows normal Python package versioning: the current major release receives bug fixes and security fixes.

Version 1.x remains available on PyPI for Python 2.7 and older Python 3 users, but it is not an actively developed branch. If you are able to run a supported Python interpreter, use the latest `checkenv` 2.x release.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability. Use GitHub's private vulnerability reporting flow for this repository if it is available, or email the maintainer listed in `pyproject.toml`.

Include the affected version, Python version, operating system, a minimal reproduction, and any relevant logs. You should receive an initial response within a week.
